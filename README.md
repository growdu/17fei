# 17fei.fun

> 情侣情趣飞行棋 + 姿势卡牌 + AI 伴侣 · Fresh + Preact + Deno

## 访问

https://17fei.fun

## 模块

- **飞行棋** — 经典双人对战 SPA
- **任务卡牌** — 9 版本翻牌 (恋爱/热恋/同居/进阶/私密/SM/户外/女仆/男仆)
- **姿势卡牌** — 3D 翻牌 + 收藏
- **姿势大全** — 100+ 姿势图鉴 + 搜索 + 收藏排序
- **AI 伴侣** — 排队订阅中
- **主题皮肤** — 5 套 (浪漫/简约/活泼/高端/暗夜)
- **VIP 激活** — 服务端 Deno KV 校验

## 技术

- **PWA** — 完整 Service Worker + 离线页 + 安装提示
- **SEO** — 13 页面独立 meta + og/twitter/json-ld + sitemap + robots
- **a11y** — skip-link + focus ring + prefers-reduced-motion + 高对比 + 强制色
- **i18n** — zh-CN, lang 标记

## 运行

```
deno task start
```

需 Deno 1.40+。

## 部署

`deno task build` 后通过 Deno Deploy 同步仓库。

## 键盘

`h` 首页 · `c` 任务 · `p` 姿势 · `g` 飞行棋 · `a` 关于 · `t` 主题 · `?` 帮助
